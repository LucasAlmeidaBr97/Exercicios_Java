# Exercícios de Java

Bem-vindo ao meu repositório de prática de Java!

Este espaço é destinado ao armazenamento de exercícios simples e projetos pequenos que desenvolvo para praticar e fixar conceitos da linguagem Java.

## Objetivo

O objetivo principal deste repositório é servir como um histórico de aprendizado e prática pessoal.

> **Nota:** Este repositório **não** tem como foco a amostragem de projetos complexos ou produtos finais para portfólio. Trata-se de um ambiente de estudos e experimentação constante.

## Conteúdo

Aqui você encontrará diversos exercícios cobrindo diferentes aspectos do Java, desde lógica de programação básica até conceitos de Orientação a Objetos.
